# Value Stream — Trigger → Insight → Action  
Human Clarity Framework (TOGAF Phase B)

This value stream describes the end‑to‑end flow of how a human moves from an emotional or situational trigger → gains clarity → takes intentional action.  
It is aligned with TOGAF Business Architecture and ArchiMate Value Stream modeling.

---

# Level 0 — Human Clarity Value Stream
**Trigger → Awareness → Insight → Decision → Action → Outcome**

This represents the universal flow of human clarity and behavior.

---

# Level 1 — Stages of the Value Stream

## 1. Trigger
An internal or external event that activates an emotional, cognitive, or behavioral response.

**Examples:**
- Someone’s tone feels disrespectful  
- A sudden change at work  
- A personal failure  
- A memory or thought loop  

**Capabilities Used:**
- Emotional Awareness  
- Trigger Navigation  

---

## 2. Awareness
The ability to notice what is happening internally and externally.

**Examples:**
- “I feel anxious.”  
- “I’m getting irritated.”  
- “This situation feels familiar.”  

**Capabilities Used:**
- Emotional Awareness  
- Pattern Recognition  

---

## 3. Insight
Understanding the meaning behind the trigger and awareness.

**Examples:**
- “This reminds me of past rejection.”  
- “I’m afraid of losing control.”  
- “This conflict is about boundaries.”  

**Capabilities Used:**
- Cognitive Reframing  
- Critical Thinking  
- Identity Clarity  

---

## 4. Decision
Choosing a response based on clarity, values, and long‑term wellbeing.

**Examples:**
- “I will pause before responding.”  
- “I will communicate my boundary.”  
- “I will not escalate this conflict.”  

**Capabilities Used:**
- Decision Clarity  
- Goal Alignment  
- Impulse Control  

---

## 5. Action
Taking intentional, value‑aligned behavior.

**Examples:**
- Speaking calmly  
- Walking away  
- Asking for clarification  
- Setting a boundary  

**Capabilities Used:**
- Adaptive Behavior  
- Communication Clarity  
- Conflict Navigation  

---

## 6. Outcome
The result of the action, feeding back into future patterns.

**Examples:**
- Conflict de‑escalates  
- Emotional stability increases  
- Relationship improves  
- Self‑respect strengthens  

**Capabilities Used:**
- Emotional Recovery  
- Long‑Term Stability  
- Relationship Maintenance  

---

# Value Stream Notes
- This value stream is **technology‑agnostic** and **process‑agnostic**, consistent with TOGAF.  
- It maps directly to the capabilities defined in Capability Map v2.  
- It will be used to derive the **Information Model** and **Reference Architecture**.  
- It represents the universal human clarity journey across all cultures and contexts.
