# Pattern Library — Emotional, Cognitive, Behavioral  
Human Clarity Framework  
TOGAF Repository — Reference Library

This Pattern Library defines the recurring emotional, cognitive, and behavioral patterns that influence human clarity.  
Patterns are reusable building blocks that help identify, diagnose, and transform human responses.

Each pattern includes:
- Pattern Name  
- Pattern Type  
- Description  
- Triggers  
- Symptoms  
- Consequences  
- Break Strategy  
- Related Capabilities  

---

# 1. Emotional Patterns

## 1.1 Overwhelm Spiral
**Type:** Emotional Pattern  
**Description:** Emotional intensity rises faster than the ability to regulate it.  
**Triggers:** Too many tasks, sudden change, conflict, sensory overload  
**Symptoms:** Panic, shutdown, irritability, racing heart  
**Consequences:** Poor decisions, withdrawal, emotional exhaustion  
**Break Strategy:** Slow breathing, grounding, reduce input  
**Related Capabilities:** Emotional Regulation, Emotional Awareness  

---

## 1.2 Rejection Sensitivity Loop
**Type:** Emotional Pattern  
**Description:** Neutral or ambiguous signals are interpreted as rejection.  
**Triggers:** Silence, delayed responses, criticism  
**Symptoms:** Anxiety, overthinking, emotional pain  
**Consequences:** Conflict escalation, avoidance, self‑doubt  
**Break Strategy:** Reality checking, value anchoring  
**Related Capabilities:** Trigger Navigation, Identity Clarity  

---

## 1.3 Anger Escalation Cycle
**Type:** Emotional Pattern  
**Description:** Anger intensifies rapidly due to perceived threat or disrespect.  
**Triggers:** Boundary violations, unfairness, tone  
**Symptoms:** Tension, raised voice, impulsive reactions  
**Consequences:** Relationship damage, regret, escalation  
**Break Strategy:** Pause, step away, clarify needs  
**Related Capabilities:** Impulse Control, Conflict Navigation  

---

# 2. Cognitive Patterns

## 2.1 Catastrophic Thinking
**Type:** Cognitive Pattern  
**Description:** Automatically imagining the worst‑case scenario.  
**Triggers:** Uncertainty, conflict, unexpected change  
**Symptoms:** Anxiety, paralysis, over‑planning  
**Consequences:** Stress, avoidance, poor decisions  
**Break Strategy:** Probability check, reframing  
**Related Capabilities:** Critical Thinking, Cognitive Reframing  

---

## 2.2 Over‑Identification
**Type:** Cognitive Pattern  
**Description:** Treating thoughts and emotions as absolute truth.  
**Triggers:** Strong emotions, past trauma, identity confusion  
**Symptoms:** “This is who I am,” “This always happens”  
**Consequences:** Stuck patterns, low flexibility  
**Break Strategy:** Distancing (“I am noticing the thought…”)  
**Related Capabilities:** Identity Clarity, Pattern Recognition  

---

## 2.3 Looping Analysis (Overthinking)
**Type:** Cognitive Pattern  
**Description:** Repetitive mental analysis without resolution.  
**Triggers:** Ambiguity, fear of mistakes, perfectionism  
**Symptoms:** Rumination, indecision, mental fatigue  
**Consequences:** Delayed action, stress, burnout  
**Break Strategy:** Timeboxing, decision rules  
**Related Capabilities:** Decision Clarity, Goal Alignment  

---

# 3. Behavioral Patterns

## 3.1 Avoidance Loop
**Type:** Behavioral Pattern  
**Description:** Avoiding discomfort instead of addressing the root issue.  
**Triggers:** Conflict, fear of failure, emotional intensity  
**Symptoms:** Procrastination, withdrawal, silence  
**Consequences:** Unresolved issues, increased anxiety  
**Break Strategy:** Micro‑actions, boundary setting  
**Related Capabilities:** Adaptive Behavior, Boundary Setting  

---

## 3.2 Escalation Pattern
**Type:** Behavioral Pattern  
**Description:** Responding to conflict with increasing intensity.  
**Triggers:** Criticism, disrespect, emotional threat  
**Symptoms:** Raised voice, defensiveness, blame  
**Consequences:** Relationship damage, emotional harm  
**Break Strategy:** Pause, clarify intent, de‑escalation  
**Related Capabilities:** Conflict Navigation, Communication Clarity  

---

## 3.3 People‑Pleasing Cycle
**Type:** Behavioral Pattern  
**Description:** Prioritizing others’ needs to avoid conflict or rejection.  
**Triggers:** Disapproval, tension, emotional discomfort  
**Symptoms:** Saying yes too often, resentment, exhaustion  
**Consequences:** Burnout, identity loss, boundary collapse  
**Break Strategy:** Values alignment, assertive communication  
**Related Capabilities:** Boundary Setting, Identity Clarity  

---

# 4. Pattern Interactions (Cross‑Pattern Dynamics)

- **Overwhelm Spiral + Looping Analysis**  
  → Emotional overload + cognitive overload = paralysis  

- **Rejection Sensitivity + People‑Pleasing**  
  → Fear of rejection drives over‑accommodation  

- **Anger Escalation + Escalation Pattern**  
  → Emotional + behavioral escalation = conflict explosion  

- **Over‑Identification + Avoidance Loop**  
  → “This is who I am” → avoids change  

These interactions help identify compound patterns in real situations.

# 5. Pattern Transformation Framework

Each pattern transformation follows:

- Trigger → Pattern Activation → Awareness → Insight → Break Strategy → New Behavior → Reinforcement


This aligns with the Value Stream and Target State Architecture.
